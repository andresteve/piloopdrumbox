Detailed Description

This class control a looper station. It is composed by:

  • Drumpad to play sounds (Keypad object)
  • Trackpad to control loop tracks (Keypad object)
  • Mutekey to mute loop tracks (Key object)
  • TFT screen and encoder (TFT object)
  • Neopixel leds (CRGB object)

void Looper::updateTrackState ( uint8_t *  msg)

Update loop track state using data received from Raspberry.

Parameters
msgThe message is 3 byte long. If msg[0] == 0:STATUS msg[1]: 0:CLEAR_REC 1: START_REC 2:STOP_REC 3:START_OVERDUB 4:STOP_OVERDUB 5:WAIT_REC 6:MUTE_REC msg[2]: loop track nummber (1-8)

If msg[0] == 1:COUNTER msg[1]: actual metronome count msg[2]: metronome max value
